# IMGW Weather Reporter - Wydania

Publiczne repozytorium wydań dla aplikacji **IMGW Weather Reporter** - asystenta pogodowego dla lotnictwa.

Służy ona do automatycznego pobierania lotniczych danych meteorologicznych bezpośrednio ze stron **IMGW-PIB (Awiacja)** oraz generowania estetycznych, zbiorczych raportów w formacie PDF.

## 🚀 Pobieranie najnowszej wersji

Zawsze aktualną wersję instalatora dla systemu Windows (x64) pobierzesz w sekcji wydań:

👉 **[Pobierz IMGW Weather Reporter (Najnowsza Wersja)](https://github.com/konkretny/imgw-reporter-releases/releases/latest)**

---

## 📋 Główne Funkcje

*   **Pobieranie Danych Pogodowych:**
    *   Aktualne komunikaty **METAR** i **TAF** dla wybranego lotniska w Polsce (kod ICAO).
    *   Prognoza obszarowa **GAMET** dla wybranego sektora (A1 – A5).
    *   Mapa **SIGWX PL**.
*   **Harmonogram Cykliczny:** Opcjonalne automatyczne pobieranie raportu codziennie o wybranej godzinie (z możliwością włączenia/wyłączenia).
*   **Generowanie PDF:** Zapisywanie gotowych raportów w wybranym przez Ciebie folderze na dysku.
*   **Autostart:** Możliwość automatycznego uruchamiania aplikacji wraz ze startem systemu Windows.

## ⚠️ Ostrzeżenie i Nota Prawna

Raporty generowane przez program mają charakter wyłącznie poglądowy i informacyjny. Dane są pobierane z publicznych serwisów IMGW-PIB. Program nie jest oficjalnym oprogramowaniem IMGW-PIB.

**Kategorycznie zabrania się wykorzystywania raportów jako oficjalnych dokumentów operacyjnych do planowania i wykonywania lotów.** Wszystkie wygenerowane pliki PDF oraz interfejs aplikacji są oznaczone klauzulą **"Nie do użytku operacyjnego"**. Korzystasz z oprogramowania na własną odpowiedzialność.
